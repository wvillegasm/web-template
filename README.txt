---
    Recommended CSS code order
- From General to the particular Top-Down
- Reset code
- Label selectors - Atoms
- ID selector if they exists
- * Structure classes Markups and Visualization, (Molecules)
- * Specific object classes: buttons, links, text, items, img, etc (Organisms)
- * Specific components classes: Logo, list, forms, tables, menus, accordions, panels, widgets (Organisms)
- * Specific sections classes: Headers, footers, navigation, main contents, galleries, etc (Organisms)
- * Specific content page classes: Login page, Home Media gallery, contact forms (Templates and pages)
- * Media queries (Mobile first, and follow atom design)


---
    CSS Attributes order
- Position: position, top, left, right, bottom, clear, z-index
- Box Model: display, float, margin, padding, width, height
- Fonts: font-family, font-size, line-height, color, text-align
- Visual: background-color, border, border-radius, outline, opacity
